可扩展 标记 语言

可扩展的：和html对比。
    html的标签的功能是开发商封装固定好的我们直接使用就可以了。xml是自定义的标签，例如：<abc>
    html是浏览器用来解释渲染成画面的(显示数据)，xml是用来(传输数据)的。
    html和xml都是用标签封装数据的，html封装后由浏览器解释渲染html标签所封装的数据。
    xml封装数据在系统与系统之间传输，不做前台显示的功能。

xml的约束

  • DTD
    •     用来编写xml的语法，例如某引入了某dtd约束文件的xml文件中只可以使用.dtd文件所指定的标签对象，
    并且标签的顺序也得严格按照.dtd文件中声明的顺序来写。
    详情参照xml文件。
    <!ELEMENT name type >
    type:(#PCDATA)  EMPTY   ANY   (子元素)
    字元素之间的关系: ,和| ：, 要求顺序一致，|只能多选一出现。
    子元素出现次数：+(1,>1) ，*(任意)，?(0,1)
    属性：Attribute:
    <!ATTLIST elementName(元素名称) 属性名称 属性类型 属性约束>
    <!ATTLIST name
	color CDATA #REQUIRED
    >
    属性类型：CDATA、枚举（a|b）多选一。ID,不能是纯数字。
    属性约束：#REQUIRED 属性必须出现,#IMPLIED 可选出现,#FIXED "值",固定值。
    实体的定义：
	<!ENTITY 别名 值 >
	元素引用 &别名; 得到值
